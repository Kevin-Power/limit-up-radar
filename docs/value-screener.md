# 長期價值選股（合理價值 / 安全邊際）

依「連續高 ROE + 穩定獲利 + 淨值法估值」的存股規則，自動掃描台股、算出合理價值與安全邊際價。

## 選股規則

| # | 條件 | 實作 |
|---|---|---|
| 1 | 連續 5 年 ROE > 15% | 視窗內每一年逐年檢查（非平均） |
| 2 | 連續 5 年稅後淨利（歸屬母公司）> 5 億 | 同上逐年檢查 |
| 3 | 最合理價值 = 最新年度每股淨值 × 五年平均ROE × 最合理本益比 | 見下方定義 |
| 4 | 安全邊際價 = 合理價值 ÷ 1.2 | |
| 5 | 慢慢買 | 現價 ≤ 安全邊際價才分批進場（報表提供參考） |
| 6 | 參考股票及現金股利 | 報表附現金殖利率欄 |
| 7 | +20% 停利、不設停損（除非跌出條件） | 操作面，報表提示 |

**名詞定義（以 2026/6 為例，最新完整年度＝2025，視窗 2021–2025）：**

- **最新年度每股淨值**：取「最新一個完整年度（2025）」年底每股淨值，**非最近一季**。
  - 每股淨值 = 母公司權益 ÷ 在外股數；在外股數 = 普通股股本 ÷ 面額（預設 10 元）。
- **最合理 ROE**：過去五年平均 ROE。
  - 逐年 ROE = 歸屬母公司淨利 ÷ 母公司權益（預設用期初、期末平均權益）。
- **最合理本益比** =（近五年「每年最高本益比」的平均 ＋ 近五年「每年最低本益比」的平均）÷ 2。
  - 由每日本益比資料，逐年取當年最高/最低（只算 PER>0 的交易日，排除虧損年）。
- **便宜價 / 合理價值 / 昂貴價** = 每股淨值 × 平均ROE ×（最低PE均 / 最合理PE / 最高PE均）。

> 年度界定：台股年報依規定隔年 3/31 前公布完畢。故 4 月（含）後最新完整年度＝去年，1–3 月＝前年。程式以 `default_latest_year()` 自動推算，亦可用 `--latest-year` 覆寫。

## 資料來源

[FinMind](https://finmindtrade.com/) 開放 API（免費）：

- `TaiwanStockInfo` — 股票清單 + 產業別
- `TaiwanStockFinancialStatements` — 綜合損益表（稅後淨利）
- `TaiwanStockBalanceSheet` — 資產負債表（母公司權益、普通股股本）
- `TaiwanStockPER` — 每日本益比 / 股價淨值比 / 殖利率
- `TaiwanStockPrice` — 每日收盤價（現價）

註冊免費帳號取得 token，額度較高、較不易被限流。

## 執行

```bash
pip install -r scraper/requirements.txt
export FINMIND_TOKEN="你的token"          # 可選但建議

# 只掃自選清單（最快）
python -m scripts.run_value_screen --codes 2330,2912,5904,1264,2395,5274,2207

# 只掃 0050 權值股
python -m scripts.run_value_screen --heavyweight

# 掃整個市場（量大、慢；建議搭配 token）
python -m scripts.run_value_screen --limit 0
```

參數：`--years`（連續年數，預設 5）、`--roe-min`（預設 15）、`--ni-min`（預設 5 億）、
`--latest-year`（覆寫最新年度）、`--limit`（掃描檔數上限，0=全部）。

## 輸出

- SQLite 資料表 `value_screen`（`data/stocks.db`）
- `data/value-screen.json`（結構化，可供前端 `/screener` 使用）
- `data/value-screen.md`（人類可讀報表）

報表欄位：平均ROE、最低ROE、最新淨值、最合理PE、便宜價、合理價值、安全邊際價、現價、折溢價%、殖利率%。
折溢價% =（合理價值/現價 − 1）×100，正值代表現價低於合理價值（偏便宜）。

### 報表格式範例（合成數據，僅示意）

| 代號 | 名稱 | 平均ROE% | 最新淨值 | 合理PE | 便宜價 | 合理價值 | 安全邊際價 | 現價 | 折溢價% |
|---|---|---|---|---|---|---|---|---|---|
| 9001 | 範例股 | 20.0 | 100.00 | 15.0 | 200.00 | 300.00 | 250.00 | 250.00 | 20.0 |

## 注意事項與限制

- **面額假設**：每股淨值用面額 10 元換算股數；少數非 10 元面額個股需自行調整 `par_value`。
- **ROE 定義**：用「歸屬母公司淨利 ÷ 母公司平均權益」；與部分網站用「期末權益」或「稅後淨利合計」會略有差異。
- **金融股**：權益/獲利結構特殊，多數 ROE 不到 15%，且淨值法估值參考性較低。
- **科目比對**：以 FinMind 中文科目名（`origin_name`）關鍵字比對，若某檔因財報格式特殊抓不到科目，會被標記原因並略過，不影響整批。
- 本工具依公開財報計算，**僅供研究，非投資建議**。
- 沙箱限制：Claude 雲端環境對外網路受限（只能連 GitHub），無法在線上即時抓 FinMind；請於本機或自有排程執行。核心計算邏輯已由 `scraper/tests/test_value_screener.py` 單元測試覆蓋。
